This is the first version of a neural network made from scratch using numpy and tensorflow (for loading and training purposes only).
It is a feed-forward neural network featuring one hidden layer only, 784 features, 128 neurons, and 10 outputs. 
It has a loss Cross-Entropy Catagorical Loss function, and uses ReLU for the activation function.
It was trained using the following hyperparemeters: Epoch: 50, Batch Size: 16, Learning Rate: 0.05

The model stats are the following:
Epoch 50, loss: 0.106, accuracy: 0.964
Test Accuracy: 0.872

There is no doubt that the model is overfit - but this is a simple neural network.
I will continue to update this project and add more features to deal with classification tasks more suitably.
